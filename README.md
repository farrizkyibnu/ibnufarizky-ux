# ibnufarizky-ux

A personal web portfolio — developed with Claude.  
Built with **HTML5**, **CSS3**, and **Vanilla JavaScript**.

## Stack

- HTML5 & CSS3 (no framework)
- Vanilla JavaScript
- Three.js (hero 3D element)
- Google Fonts — EB Garamond + Poppins

## Features

- Custom cursor with magnetic buttons
- Hero entrance animation (staggered reveal)
- Mouse parallax on background glow orbs
- Scroll-triggered section animations
- Animated stats counter
- Responsive layout
